# PerlinNoiseGenerator

Prototipo para generar mapas de ruido, visualizarlos en WinForms y exportarlos como archivos CSV.

### Uso:

Ejecutar la aplicación para ver una ventana con un mapa generado. Presionar la barra espaciadora generará un nuevo mapa. Cada mapa visualizado se exportará automaticamente al directorio bin/Debug/Exports.

La aplicación solo es compatible con Windows.
